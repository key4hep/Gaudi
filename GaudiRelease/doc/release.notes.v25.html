Gaudi v25r0 (2014-02-14)

Externals version: LCGCMT_66

General Changes

  • This is the first version of Gaudi that can be built with both ROOT 5.x and ROOT 6.x.
    All the packages have the major version increased to reflect the change.
  • Replaced Reflex Plugin Service with a new Reflex-free and ROOT-free Plugin Service.
    (Marco Clemencic)
  • Several improvements to the CMake configuration.
    (Marco Clemencic, Hubert Degaundenzi)
  • Added configuration files for the hwaf system.
    (Sebastien Binet)

Packages Changes

  • GaudiPolicy (v14r0):
    • Prevent path normalization in test arguments if they do not point to a file. Needed because sometimes an argument is a URL and it must not be normalized.

      (Marco Clemencic)
    • Fixed LBCORE-292: added project and slot details to the test summaries
