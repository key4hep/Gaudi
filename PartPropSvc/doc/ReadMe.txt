-----------------------------------------------------------------------------
Usage:
-----------------------------------------------------------------------------

JobOptions:
   (string) "InputFile": name and type of input format. Name is one of:
                         "PDG", "Herwig", "Pythia", "IsaJet"
                         default: "PDG"

                 format: "FILE"
			 "FILE=TYPE"
                         "FILE1=TYPE1, FILE2=TYPE2, FILE3=TYPE3"

Accessors: see GaudiKernel/IPartPropSvc.h

                                      generated from input files read
                                      in.

				  const std::string& ) : sets the
				  function to handle unknwon particle
				  IDs, such as heavy ions generated by G4
